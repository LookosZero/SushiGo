# SushiGo - Implementación en Java

Proyecto de desarrollo del juego **SushiGo** realizado en **Java** para la asignatura de **Algoritmos y Estructuras de Datos I (AEDI)**.

## Descripción

Este proyecto consiste en la implementación del juego de cartas *SushiGo*, centrado en aplicar conceptos fundamentales de estructuras de datos y lógica algorítmica.

El sistema simula el flujo completo del juego, gestionando cartas, turnos, puntuaciones y reglas, permitiendo ejecutar partidas de forma programática.

## Objetivos

- Aplicar conceptos de:
  - **Estructuras de datos** (listas, arrays, etc.)
  - **Programación orientada a objetos (POO)**
  - **Diseño modular**
- Implementar la lógica completa de un juego real
- Gestionar estados de partida y cálculo de puntuaciones

## Características principales

- Modelado de cartas y baraja
- Gestión de jugadores
- Sistema de turnos
- Cálculo de puntuación según reglas de SushiGo
- Simulación completa de partidas

## Tecnologías utilizadas

- **Java**
- Programación orientada a objetos (POO)
